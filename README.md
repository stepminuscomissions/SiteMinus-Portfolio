# Minus Portfolio

Site de portfolio e comissões do artista digital **Minus**.

## Stack

- React 19
- Vite 6
- Tailwind CSS 4
- TypeScript 5.8
- Motion (Framer Motion)

## Funcionalities

- Galeria de artes com filtros por tags
- Sistema de comissões com preços
- Fila de projetos em tempo real
- Tema claro/escuro
- Boot screen cyberpunk
- Turntable player (Web Audio API)
- Pacman minigame

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Automático via GitHub Pages ao push no `main`.
